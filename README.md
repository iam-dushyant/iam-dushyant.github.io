
# Static Portfolio (HTML/CSS/JSON)
A lightweight, GitHub Pages–ready portfolio with: About, Projects (JSON-driven with search/filter), Resume (embed + download), Contact.

## Quick start
1. Edit `data/profile.json` (name, role, about, email, GitHub, LinkedIn, skills).
2. Edit `data/projects.json` (title, summary, tech, repo, demo).
3. Replace `assets/resume.pdf` with your real resume file.
4. Push to a GitHub repo and enable GitHub Pages (Settings → Pages → Deploy from branch).
   - User site: `<username>.github.io` repo.
   - Project site: any repo + Pages. Access via `https://<username>.github.io/<repo>/`.

No build step. Just HTML/CSS/JS.
