MNIST classification: A Computer Vision approach

The classic MNIST classification uses conventional Machine Learning algorithms to classify numbers from the MNIST dataset.

While the modern solutions often rely on deep learning, this project demonstrates how traditional image processing techniques - such as edge detection, shape analysis, and feature extraction - can still be used for digital classification.

The purpose of this project is to explore Computer Vision and enhance my skills. This small project seemed a good starting point.

Steps for implementation

  1. Data preparation:
    • Used the MNIST classification dataset of 28x28 grayscale images of handwritten digits, ranging from 0 to 9.
    • Processed each image by thresholding (converting to binary) and applying noise reduction if necessary.
  2. Feature extraction:
    • Applied Canny edge detection to highlight contours of each digit.
    • Extracted shape detectors, such as Hu Moments.
  3. Classifier
    • Created feature vectors from the extracted shape descriptors.
    • Used a simple Machine Learning classifier.
    • Trained the ML classifier onto a subset of MNIST dataset.
  4. Evaluation
    • Tested the trained classifier on a separate test set.
    • Computed accuracy and confusion matrix.
